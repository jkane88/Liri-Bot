## This is a LIRI node application. (Language Interpretation and Recognition Interface)

- A command line app that accepts user input and returns data.
  
- Uses the Spotify API, Bands in town API & OMdB API.
  
- Commands:
 - **spotify-song**
 - **concert-this**
 - **movie-this**
